* Send messages through Telegram bot


** Example config in .bashrc
   #+begin_src bash
     export TELEGRAM_BOT_TOKEN='the-token'
     export TELEGRAM_BOT_CHAT='the-chat-id'
     export TELEGRAM_BOT_PROXY=socks5://localhost:9999 # Optional
     source ~/telegram-bot-client/client.sh
   #+end_src


** Example usage

*** Explicit, via $1
    =send-telegram-message 'hello'=

*** Implicit, from stdin
    =ls -lah | send-telegram-message=
